# Projekin Ajaa — Landing Page

Landing page statis (HTML/CSS/JS murni, tanpa build step).

## Sebelum deploy
1. Buka `index.html`, cari `GANTI NOMOR WA DI SINI`, ganti `WA_NUMBER` dengan nomormu (format internasional, contoh: 6281234567890).
2. Ganti placeholder `[...]` di bagian FAQ, testimoni, dan statistik.

## Deploy ke Vercel
**Cara termudah (drag & drop):**
1. Buka https://vercel.com/new
2. Tarik folder ini ke area upload — selesai.

**Atau via CLI:**
```
npm i -g vercel
vercel
```
